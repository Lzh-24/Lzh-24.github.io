## 2. 不同工具的优缺点

| 工具/语言        | 擅长领域 (Good At)                                           | 不擅长/一般领域 (Not Good At)            |
| ---------------- | ------------------------------------------------------------ | ---------------------------------------- |
| **Linux / Bash** | 字符串处理、分析数组和矩阵、管道处理、文件处理               | 复杂的应用程序开发                       |
| **R**            | 分析数据、矩阵处理、处理文件和管道、统计学、绘图、基础机器学习 | 可能偏向于工程化实现或大型项目           |
| **Python**       | 字符串和数据处理（复杂任务）、处理文件和管道、机器学习、深度学习 | 在C/C++擅长的“速度”方面不擅长            |
| **C / C++**      | 追求速度的程序、程序调试                                     | 在快速开发或数据处理便捷性上不如Python/R |

## 3. 生物信息学的研究流程

0. Question： 基于生物、医药学等学科的知识提出一个问题

1. Information： 收集信息，统计生物学及医学方面的数据

2. Analysis: 将原始测序数据（Raw Data）转化为可用于生物学解释或机器学习建模的“干净表格”

3. Modeling: 借助概率模型以及计算算法得出结论

## 5. 算法(algorithm)和模型(model)的区别

|               | **算法**                                                     | **模型**                                                     |
| ------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **定义**      | 一个有限、确定、有效的指令序列，用于解决某一类特定问题或完成某一项计算任务。 | 从数据中学习到的、用于对未知数据进行预测或决策的数学结构及其参数集合。 |
| **可变性**    | 相对固定，是通用规则                                         | 随训练数据变化                                               |
| **数据信息**  | 不包含                                                       | 包含                                                         |
| **数学本质**  | 映射函数，计算过程                                           | 参数化函数                                                   |
| **输入/输出** | 输入：训练数据、超参数、初始参数；输出：更新后的参数（即模型）。 | 输入：新样本的特征向量；输出：预测值（分类标签、回归值、概率分布等）。 |
